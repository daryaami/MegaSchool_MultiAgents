# Веб-интерфейс MegaSchool Interview Coach

## Быстрый старт

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Запустите веб-сервер:
```bash
python run_web.py
```

3. Откройте в браузере: http://localhost:5000

## Использование

1. **Заполните форму** с данными кандидата:
   - Название команды
   - Имя кандидата
   - Позиция (например, "Backend Developer")
   - Грейд (Junior/Middle/Senior)
   - Опыт работы

2. **Нажмите "Начать интервью"**

3. **Отвечайте на вопросы** интервьюера в текстовом поле

4. **Просматривайте результаты**:
   - Вопросы интервьюера отображаются в синих сообщениях
   - Ваши ответы отображаются в зелёных сообщениях
   - Внутренние мысли Observer и Interviewer отображаются в серых сообщениях с метками [Observer] или [Interviewer]

5. **Завершите интервью**:
   - Нажмите кнопку "Завершить" или введите "стоп" в поле ответа
   - После завершения отобразится финальный отчёт с результатами
   - В финальном отчёте есть кнопка "Начать заново" для запуска нового интервью

## Финальный отчёт

Отчёт включает:
- **Вердикт**: рекомендация (Hire/No Hire/Strong Hire), грейд, уверенность
- **Технический обзор**: подтверждённые навыки, пробелы в знаниях, детали по темам
- **Soft Skills**: ясность, честность, вовлечённость
- **Персональный Roadmap**: рекомендации по улучшению навыков

## Особенности

- **Адаптивная сложность**: система автоматически подстраивает сложность вопросов
- **Скрытая рефлексия**: внутренние мысли Observer сохраняются для анализа
- **Автоматическое сохранение**: результаты сохраняются в `logs/interview_log_{session_id}.json`

## Технические детали

- Веб-интерфейс использует Flask для backend
- Коммуникация происходит через WebSocket-подобный polling механизм
- Каждое интервью запускается в отдельном потоке
- Сообщения обрабатываются асинхронно через asyncio очереди
