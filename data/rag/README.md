# RAG Data Directory

Эта директория предназначена для хранения файлов базы данных RAG (Retrieval-Augmented Generation).

## Необходимые файлы

Для работы RAG модуля необходимо поместить в эту директорию следующие файлы:

1. **faiss_index.bin** - FAISS индекс для быстрого поиска по эмбеддингам
2. **data.pkl** - Данные с вопросами и ответами в формате pickle (pandas DataFrame)

## Структура данных

Файл `data.pkl` должен содержать pandas DataFrame со следующими колонками:
- `Category` - категория вопроса
- `Skill` - навык
- `Level` - уровень сложности
- `Question` - текст вопроса
- `Answer` - ожидаемый ответ
